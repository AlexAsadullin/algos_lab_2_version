## Алгоритмы
- два указателя (`two_pointers.py`)
- двоичный поиск (`binsec.py`)
- экспоненциальный поиск (`exponential.py`)
- рекурсивное деление диапазона (`binary_divide.py`)

## Датасеты

Все датасеты представляют собой пары отсортированных массивов `m` и `n` с различной четностью и длиной.

- **Dataset 1**
  - `m`: 5 млн чётных чисел, отсортированных по возрастанию.
  - `n`: 10 млн нечётных чисел, все элементы больше любого элемента `m` (гарантированно нет пересечений).
- **Dataset 2**
  - `m`: 5 млн чётных чисел (короткий массив).
  - `n`: 10 млн нечётных чисел (длинный массив).
  - В конце массивов добавлена общая точка пересечения.
- **Dataset 3**
  - `m`: 10 млн нечётных чисел (длинный массив).
  - `n`: 5 млн чётных чисел (короткий массив).
  - В конце массивов добавлена общая точка пересечения.

Во всех датасетах:
- элементы m и n имеют разную четность, при этом четность внутри массива гарантированно одинаковая
- гарантируется, что для основной части данных `m[i] < n[i]`.

## Структура

- `main.py` — генерация тестовых датасетов, запуск всех алгоритмов, сбор метрик, построение графиков Plotly.
- `two_pointers.py` — алгоритм два указателя.
- `binsec.py` — алгоритм пересечения на основе двоичного поиска.
- `exponential.py` — алгоритм пересечения с экспоненциальным поиском.
- `binary_divide.py` — алгоритм пересечения через двоичное деление диапазона.
- `requirements.txt` — зависимости проекта.

## Зависимости

- Python 3 (современная версия, например 3.10+)
- Основные библиотеки:
  - `plotly`
  - `psutil`

Установка:

- `pip install -r requirements.txt`

## Запуск

- Запустить скрипт:
  - `python main.py`
- Скрипт:
  - создаёт большие тестовые массивы (до десятков миллионов элементов)
  - запускает каждый алгоритм на 3 наборах данных
  - измеряет среднее время и использование памяти
  - сохраняет HTML‑графики в подпапку `charts/`.
